# Face Recognition Attendance System

A web-based attendance system that uses face recognition to automatically mark attendance in real time.

## Features

* Real-time face detection and recognition
* Automatic attendance marking
* Duplicate entry prevention
* User registration with webcam

## Tech Stack

* Python
* Flask
* OpenCV
* NumPy, Pandas
* Scikit-learn

## Run

```bash
python app.py
```
